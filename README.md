# Initializtion
## You'll need concurrently to run this
``` npm install concurrently -g
```
## You'll also need to have mongodb running in the background
```
mongod
```
## After all that you'll need to install the dependancies for both packages.json
```
npm install
cd client
npm install
```
# Start up
```
npm run start
```
